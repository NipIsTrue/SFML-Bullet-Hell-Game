Getting Started
============================================
To start, run the command ``git clone https://github.com/NipIsTrue/SFML-Bullet-Hell-Game``

.. image:: /images/clone.png

Then, using your file explorer, navigate to the directory :file:`{Installation Directory}/bin/Release/` and run the executable named 'Bullet Hell Wave Game'

.. image:: /images/exedir.png

The game should launch normally and you should see something like this:

.. image:: /images/game.png

If the game fails to launch, check out :doc:`common_problems`
